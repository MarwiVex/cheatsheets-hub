# Mathematical and Statistical Cheat Sheets Hub
# Repositorio de Formularios Matemáticos

---

## English Overview
This repository serves as a central hub for a collection of quick-reference mathematical, statistical, and engineering guides I have compiled. All cheat sheets are typeset in LaTeX to provide clean, easy-to-read formats for faster problem-solving. 

Each guide below is an independent repository containing both English and Spanish versions, along with their respective LaTeX source codes.

### The Collection

* [**Probability and Statistical Inference**](./probability-inference-cheatsheet)
    Covers everything from basic combinatorics and Kolmogorov Axioms to advanced stochastic processes, Markov Chains, and Bayesian inference.
* [**Ordinary Differential Equations (ODEs)**](./differential-equations-cheatsheet)
    A quick reference for identifying and solving various types of differential equations, including first/second-order techniques, systems, and Laplace Transforms.

*(More topics will be added as they are developed)*

## Cloning this Hub / Clonar este Repositorio

Because this repository uses Git Submodules to link the individual cheat sheets, you need to clone it recursively to download all the linked content:

```bash
git clone --recursive [https://github.com/MarwiVex/cheatsheets-hub.git](https://github.com/MarwiVex/cheatsheets-hub.git)
```
*(Si ya lo clonaste sin este comando, puedes inicializar los submódulos usando: `git submodule update --init --recursive`)*
